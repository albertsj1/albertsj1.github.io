It's been quite a while since my last real post. Let's just ignore that ['Now with more Hugo'][1] post. :) I've always wanted to write about all the cool things I'm working on in my blog, but there's just never enough time in the day.

Although I don't care for the name 'Milk Music', the app is really nice. There is a native app for all recent Samsung phones, tablets, TVs and some other misc devices.  You can also listen on your computer via [your web browser][3].

I've been working on a lot of different technologies lately, such as: Cassandra, Hadoop, Elasticsearch, Kibana, Sensu, ... Hopefully I'll find some time to write about them soon. :)
