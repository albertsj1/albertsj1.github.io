[Wetpaint | Wiki - Create a free wiki. Many looks. No code required.](http://www.wetpaint.com/)

This looks like a really nice site as an alternative to using MySpace.  You can create web pages without using any code at all and the pages look pretty good.  Best of all, it's FREE!

I did notice one problem though.  I took a look at one persons site and it let me edit their site.  Hopefully there are options to disable that.[![wetpaintsm.jpg](http://www.mindlesstechie.net/blog/wp-content/uploads/2007/01/wetpaintsm1.jpg)](http://www.wetpaint.com)
